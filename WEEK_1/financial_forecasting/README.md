# Financial Forecasting

## Objective
Implement a recursive algorithm to predict future values based on past growth rates.

## What is Recursion?
Recursion is a technique where a method calls itself to solve a problem. Every recursive method must have:
- A base case to stop recursion.
- A recursive case that reduces the problem size.

## Time Complexity
- Time Complexity: O(n)
- Space Complexity: O(n) due to the recursion call stack.

## Optimization
The recursive solution can be optimized by:
- Using iteration to avoid recursion overhead.
- Using the direct mathematical formula for compound growth when applicable, reducing computation.
