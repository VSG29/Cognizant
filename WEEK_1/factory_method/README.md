# Factory Method Pattern

## Objective
Create different types of documents using the Factory Method Design Pattern.

## Classes
- Document.java
- WordDocument.java
- PdfDocument.java
- ExcelDocument.java
- DocumentFactory.java
- WordFactory.java
- PdfFactory.java
- ExcelFactory.java
- FactoryTest.java

## Design Pattern Used
Factory Method Pattern

## Advantages
- Encapsulates object creation.
- Reduces coupling.
- Makes code easier to extend.
- Follows the Open/Closed Principle.

## Output

Opening Word Document
Opening PDF Document
Opening Excel Document
