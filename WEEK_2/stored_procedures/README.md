# Stored Procedures

## Scenario 1
Creates a stored procedure `ProcessMonthlyInterest` that updates the balance of all savings accounts by applying 1% monthly interest.

## Scenario 2
Creates a stored procedure `UpdateEmployeeBonus` that increases employee salaries by a bonus percentage for a specified department.

## Scenario 3
Creates a stored procedure `TransferFunds` that transfers money from one account to another after checking that the source account has sufficient balance.

## Concepts Used
- Stored Procedures
- Parameters
- UPDATE
- IF Condition
- COMMIT
- DBMS_OUTPUT.PUT_LINE
