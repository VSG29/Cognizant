# JUnit Setup

## Objective
Set up JUnit 4.13.2 using Maven.

## Tools Used
- Java
- Maven
- JUnit 4.13.2
- IntelliJ IDEA

## Dependency

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

## Result

JUnit was successfully configured and a sample test executed successfully.
