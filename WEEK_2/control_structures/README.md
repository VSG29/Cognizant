# Control Structures (PL/SQL)

## Scenario 1
Applies a 1% discount to loan interest rates for customers older than 60 years.

## Scenario 2
Promotes customers with a balance greater than 10000 to VIP status.

## Scenario 3
Prints reminder messages for loans that are due within the next 30 days.

## Concepts Used
- FOR Loop
- IF Condition (through WHERE clause)
- UPDATE Statement
- DBMS_OUTPUT.PUT_LINE
- COMMIT
