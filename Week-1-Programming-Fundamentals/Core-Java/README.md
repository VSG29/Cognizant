# Core Java

## Overview

Core Java is the foundation of Java programming. It includes the basic concepts required to develop Java applications such as variables, operators, control statements, object-oriented programming, exception handling, collections, and multithreading.

## Topics Covered

### 1. Introduction to Java

* Features of Java
* JVM, JRE and JDK
* Java Program Structure

### 2. Variables and Data Types

* Primitive Data Types
* Non-Primitive Data Types
* Variable Declaration
* Type Casting

### 3. Operators

* Arithmetic Operators
* Relational Operators
* Logical Operators
* Assignment Operators
* Bitwise Operators

### 4. Control Statements

* if, if-else
* switch
* for loop
* while loop
* do-while loop

### 5. Arrays

* One-Dimensional Arrays
* Two-Dimensional Arrays
* Enhanced for Loop

### 6. Strings

* String
* StringBuilder
* StringBuffer
* String Methods

### 7. Object-Oriented Programming

* Classes and Objects
* Constructors
* Inheritance
* Polymorphism
* Abstraction
* Encapsulation

### 8. Exception Handling

* try
* catch
* finally
* throw
* throws

### 9. Collections Framework

* List
* Set
* Map
* Queue
* Stack

### 10. Multithreading

* Thread Class
* Runnable Interface
* Synchronization
* Thread Lifecycle

## Learning Outcome

After completing this module, I will be able to write Java programs, apply object-oriented programming concepts, use Java collections effectively, and develop efficient applications using Core Java fundamentals.
