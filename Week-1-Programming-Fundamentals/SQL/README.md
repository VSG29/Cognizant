# SQL

## Overview

Structured Query Language (SQL) is used to store, retrieve, modify, and manage data in relational databases.

## Topics Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* JOINS
* Aggregate Functions
* Subqueries
* Constraints
* Normalization

## Common SQL Operations

* Insert Data
* Update Data
* Delete Data
* Retrieve Data

## Learning Outcome

After completing this module, I will be able to write SQL queries, retrieve information efficiently, and manage relational databases.
